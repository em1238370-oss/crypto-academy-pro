## ğŸ’¡ Design Philosophy

### Why "Crypto: Not Just Madness"?
The main headline reflects the project's core philosophy: cryptocurrency is often seen as chaotic and unpredictable, but with the right tools and knowledge, it can be approached systematically and professionally.

### Design Decisions

1. **Dark Theme:** Professional, modern look that reduces eye strain
2. **Purple/Green Accents:** Crypto-themed colors that stand out
3. **Animated Background:** Blockchain elements create immersive experience
4. **Modular Structure:** Each feature is a separate sub-site for easy maintenance
5. **Free + Paid Model:** Free features attract users, paid features provide value
