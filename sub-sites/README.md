# Подсайты

Здесь будут создаваться новые сайты, которые будут прикреплены к главному сайту.

## Структура

Каждый новый сайт будет в отдельной папке:
- `sub-site-1/` - первый подсайт
- `sub-site-2/` - второй подсайт
- и т.д.

## Интеграция с главным сайтом

Подсайты будут доступны через главный сайт и могут использовать общий backend.
