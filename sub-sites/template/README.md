# Шаблон для нового подсайта

Этот шаблон можно использовать для создания новых подсайтов, которые будут прикреплены к главному сайту.

## Структура

```
sub-site/
├── index.html      # Главная страница подсайта
├── styles.css      # Стили подсайта
├── scripts/        # Скрипты подсайта (если нужны)
└── assets/         # Ресурсы подсайта
```

## Интеграция

Подсайт может использовать общий backend главного сайта через API:
- `http://localhost:4000/api/...`

## Доступ

Подсайт будет доступен через главный сайт или напрямую по своему URL.
