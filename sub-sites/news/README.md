# Crypto News Site

Сайт для новостей о криптовалютах.

## Фон

Поместите фоновое изображение в папку `assets/` с именем `news-background.jpg`

**Требования к изображению:**
- Размеры: 7250px × 1750px (ширина × высота)
- Формат: JPG (высокое качество, без артефактов)
- Содержание: Темный фон с желто-оранжевыми источниками света в правом верхнем и левом нижнем углах

## Структура

- `index.html` - главная страница
- `styles.css` - стили с желтыми акцентами
- `assets/news-background.jpg` - фоновое изображение
