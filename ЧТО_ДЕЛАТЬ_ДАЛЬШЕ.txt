✅ ЧТО НУЖНО СДЕЛАТЬ:

СПОСОБ 1: Через GitHub Desktop (РЕКОМЕНДУЕТСЯ)

1. Закройте браузер (или оставьте открытым - не важно)

2. Откройте GitHub Desktop

3. Посмотрите внизу экрана - должна быть кнопка:
   "Commit X files to main" (синяя кнопка)

4. Если кнопка есть:
   - Напишите сообщение: "Исправлен заголовок Crypto Coach"
   - Нажмите "Commit X files to main"

5. После коммита появится кнопка "Push origin" вверху справа
   - Нажмите "Push origin"

6. Готово! Render обновит сайт через 1-2 минуты

═══════════════════════════════════════════════════════════

СПОСОБ 2: Через терминал (если GitHub Desktop не работает)

1. Откройте Терминал
